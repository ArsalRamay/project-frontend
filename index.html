Marketing website done with Astro

Astroship is a starter template for startups, marketing websites & landing pages. Built with Astro.build and TailwindCSS. You can quickly create any website with this starter.

Everything you need to start a website

Astro comes batteries included. It takes the best parts of state-of-the-art tools and adds its own innovations.

Bring Your Own Framework

Build your site using React, Vue, or plain HTML + JS.

100% Static HTML, No JS

Astro renders your entire page to static HTML for fast loading.

On-Demand Components

Astro automatically hydrates components only when needed.

Broad Integration

Supports TypeScript, Tailwind, Markdown, and more.

SEO Enabled

Automatic sitemaps, feeds, and pagination included.

Community

Powered by hundreds of open-source contributors.

Build faster websites.

Pull content from anywhere and serve it fast with Astro's next-gen island architecture.
